#### 1. 구독 추가 모달 - ✅ 통과

**변경 내용:** 페이지 이동 대신 헤더에서 바로 모달창으로 구독 추가

**확인 사항:**

- [x] 모달 내 폼 정상 작동
- [x] ESC 키로 모달 닫기
- [x] 바깥 클릭으로 모달 닫기
- [x] 취소 버튼으로 모달 닫기

#### 2. 종료일 자동 계산 - ✅ 통과

**확인 사항:**
- [x] 시작일 선택 시 종료일 자동 입력
- [x] 기간 변경 시 종료일 재계산

- [x] 이모지 선택 시 버튼에 표시
- [x] 선택한 이모지 DB 저장 및 캘린더 표시

---

#### 4. costLevel 색상 버그 수정 - ✅ 통과

**변경 내용:** 회당 비용 기준을 월금액 → 총금액으로 변경

| 조건 | 레벨 | 색상 |
|------|------|------|
| 회당 비용 ≤ 총금액/20 | good | 🟢 초록 |
| 회당 비용 ≤ 총금액/10 | normal | 🟡 노랑 |
| 회당 비용 > 총금액/10 | warning | 🔴 빨강 |

**테스트 결과:**

| 구독 | 총 금액 | 사용 횟수 | 회당 비용 | 예상 색상 | 실제 색상 | 결과 |
|------|---------|-----------|-----------|-----------|-----------|------|
| 12개월 구독 | ₩120,000 | 16회 | ₩7,500 | 🟡 노랑 | 🟡 노랑 | ✅ |

### 신규 기능 QA

#### 1. 구독 상세 페이지 - ✅ 통과

**접근 방법:** 대시보드에서 구독 카드 클릭 → `/subscriptions/{id}` 이동

**표시 내용 확인:**
- [x] 총 금액 표시
- [x] 월 환산 금액 표시
- [x] 총 사용 횟수 표시
- [x] 회당 비용 표시 (색상 구분 포함)
- [x] 구독 정보 (시작일)
- [x] 출석 체크 버튼 상태 표시
- [x] 최근 사용 기록 (요일 포함)
- [x] 수정/삭제 버튼

---

#### 2. 구독 수정 - ✅ 통과

**플로우:** 상세 페이지 → "수정" 버튼 → 수정 폼 → "수정하기" → 상세 페이지 복귀

**확인 사항:**
- [x] 기존 데이터 폼에 채워짐
- [x] 서비스 이름 수정 가능
- [x] 저장 후 상세 페이지 복귀

**발견된 문제:**
- ⚠️ 시작일 데이터가 수정 폼에 미리 채워지지 않음 (재입력 필요)

---

#### 3. 구독 삭제 - ⚠️ 부분 통과

**플로우:** 상세 페이지 → "삭제" 버튼 → 대시보드 복귀

**확인 사항:**
- [x] 삭제 후 대시보드 복귀
- [x] 구독 데이터 삭제됨
- [x] 통계 초기화

**발견된 문제:**
- ⚠️ 삭제 확인 다이얼로그가 작동하지 않음 (바로 삭제됨)

---

#### 4. 이모지 미리보기 - ✅ 통과

**기능:** 아이콘 드롭다운 선택 시 입력란 좌측에 이모지 표시

**확인 사항:**
- [x] 🏋️ gym 선택 시 이모지 표시
- [x] ☕ coffee 선택 시 이모지 변경
- [x] 🎬 netflix 등 다른 아이콘 정상 작동
